## Key Concepts

### Android Project Structure
- `app/src/main/java` - Main application code
- `app/src/main/res` - Resources (layouts, drawables, strings)
- `app/src/main/AndroidManifest.xml` - App configuration

### Activities and Lifecycle
- Activities represent a single screen
- Lifecycle methods: onCreate, onStart, onResume, onPause, onStop, onDestroy

### Gradle
- Build system for Android
- `build.gradle.kts` - Module-level build configuration
- `settings.gradle.kts` - Project-level settings
