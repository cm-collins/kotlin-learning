## Key Concepts

### Unit Testing
- Test individual functions/classes in isolation
- Use JUnit or Kotlin Test
- Mock dependencies for isolated testing

### Integration Testing
- Test multiple components working together
- Test database, network, and other integrations

### UI Testing
- Test user interactions and UI behavior
- Compose Testing for Compose UI
- Espresso for View-based UI
